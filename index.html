# Configuración

  • Instalación de Ruby

   * Usted contará con Ruby, RubyGems y RSpec para aprender Ruby.    * Se recomienda utilizar [RVM] (http://rvm.beginrescueend.com), pero se puede utilizar cualquier version que desea. Nos apoyamos oficialmente en Ruby 1.9 pero probablemente trabajaremos con 1.8.7 también.    * He aquí algunas instrucciones correctas[] (http://www.wiki.devchix.com/index.php?title=Workshop_Installation_Notes_Rails_3) para instalar el software necesario.    * Estas son algunas [otras buenas instrucciones] (http://ruby.railstutorial.org/ruby-on-rails-tutorial-book # sec: RubyGems), y utilizando RVM.

  • Abra una ventana del terminal. (En Mac OS X puede utilizar la app Terminal.)

  • Instale RSpec

         gem install rspec

  • Introduzca el directorio del curso. (Ese es el mismo directorio que el archivo index.html `` es publico)

         learn_ruby cd

  • Introduzca el 00_hello `laboratorio '.

         00_hello cd

  • Abra [index.html] (./00_hello/index.html) Archivo del laboratorio en un explorador web y siga las instrucciones allí.

         abierto index.html # esto sólo funciona en Mac OS X!

# El uso de estos ejercicios

Su directorio de cursos tiene una lista de directorios de laboratorios. Cada directorio tiene un archivo "spec". Vd. tendrá que escribir todo el código necesario para hacer que pase todas las especificaciones.

Para conseguir implicarse en este proceso, vaya al "hola" de laboratorio con cd 00_helloy lea las instrucciones detalladas en su archivoindex.html.

Si ha llegado a través de "hola", ¡felicidades! Ahora es el momento de ir al iguiente directorio (cuyo nombre comienza con 01_) y comenzar a aprender Ruby!

# Resumen del Curso

Cada curso es diferente, así que consulte con su instructor para obtener detalles sobre el Sylabus que él o ella hayan elegido. Aquí está un examen de algunos de los laboratorios:

Hola    Te da una idea básica de cómo utilizar estas pruebas.    Véase [00_hello/index.html] (00_hello/index.html) para obtener instrucciones mas detalladas. Temperatura    Escribir clases y hacer algo de matemáticas básicas. Calculadora introduce un poco más de matemáticas, y usted tendrá que repetirlas. Cerdo \ _latin permite manipular cadenas en un lenguaje imaginario. En \ _words pide un número para decir sí en Inglés. Extiende una clase incorporada del lenguaje. Al final, vamos a usar los condicionales, matrices de iteración, y tal vez incluso un poco de recursividad. Bloques introduce sintaxis de bloques incluyendo rendimiento

Configuración avanzada

Después de coger el ritmo de las cosas, Vd. podrá leer sobre técnicas avanzadas de configuración [] (advanced_setup.html) como se "tira" de los cambios y se bifurcan. Pero no te preocupes por eso, al principio.

Problemas? ¿Preguntas?

En primer lugar, preguntele a su compañero. Luego, pida ayuda a su instructor.

Luego, pídale ayuda a Google (en serio). Si hay un error, intente copiar la cadena del error/es y pegarlo en un cuadro de búsqueda [de Google] (http://google.com). Si eso no ayuda, haga una búsqueda en uno de estos sitios:

  • http://stackoverflow.com/
  • http://www.ruby-forum.com

También puede encontrar ayuda en el sitio [TestFirst.org] (http://testfirst.org) o [Lista de Enseñanza de mail de Enseñanza "Primero los Test"] (http://groups.google.com/group/test-first-teaching ).

Y hay muchas conferencias relacionadas, notas y videos que están en http://CodeLikeThis.com.

Recuerde, si Vd. publica una cuestión técnica en línea, Vd. debe

  1. Incluir números de versión:

         $ Ruby-v          ruby 1.8.6 (2008-08-11 patchlevel 287) [universal darwin9.0]

  1. Excepción en la que ya ha buscado una respuesta
  2. Si es posible, incluir fragmentos de código que reproducen el problema de manera aislada
